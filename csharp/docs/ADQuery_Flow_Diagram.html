ADQuery Request Flow

USER (Browser)
Natural language query:
"Show me all contractors in Finance"
▼
🔒 ON-PREMISES (Internal Network)
1
Windows Authentication
User identity verified via Kerberos/NTLM
▼
2
Build LLM Prompt
Add AD schema context, org mappings, allowed attributes
✓ Sent to LLM: Query text, column headers, format patterns
✗ NOT sent: User data, CUI, PII, actual values
🌐 EXTERNAL (Internet) — HTTPS Only
3
Send to Claude API
Query + context only
→
Claude API
Generates JSON plan
→
4
Receive JSON Plan
Declarative structure
LLM returns declarative JSON describing what to query — NOT executable code
🔒 ON-PREMISES (Internal Network)
5
Plan Validation
✓ Attributes on allowlist?
✓ Complexity within limits?
✓ Valid operations only?
REJECT if any check fails
▼
6
Execute Against AD
Translate plan to LDAP queries (values escaped)
▼
7
Process Results
Format output, save to file share, write audit log
▼
RESPONSE TO USER
Data preview in browser
Download link for full results
Audit log entry created
On-Premises (all data stays here)
External (query text only, no data)
Security Validation Gate
